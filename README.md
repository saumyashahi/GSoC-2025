## Project Overview

This repository documents the **Google Summer of Code 2025** contribution to [Sugar Labs](https://www.sugarlabs.org/), focusing on the development of the **Masonry Module** for [Music Blocks v4](https://github.com/sugarlabs/musicblocks-v4).

The **Masonry Module** is a scalable visual programming system that enables students to explore programming concepts through draggable, stackable, and nestable "bricks" that map directly to executable Abstract Syntax Trees (ASTs).
